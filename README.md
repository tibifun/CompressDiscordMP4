# CompressDiscordMP4
It NEEDS FFMPEG you NEED to have it installed

This one is also made for the AV1 encoder which is better at retaining information from less bitrate, recent GPUs should have it older ones wont support it. Change this as you see fit.


place it in C:\Users\yournamehere\AppData\Roaming\Microsoft\Windows\SendTo
You will be able to right click your file hover "Send To" click the script and it will automatically start compressing it 

It's set to 720p 60fps 

any problems send it to your favorite AI agent.
